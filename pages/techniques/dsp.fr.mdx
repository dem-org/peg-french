[Li et al., (2023)](https://arxiv.org/abs/2302.11520) propose une nouvelle technique d'incitation pour mieux guider le LLM dans la production du résumé souhaité.

Un LM à politique réglable est formé pour générer le stimulus/indice. Voir plus d'utilisation de RL pour optimiser les LLMs.
